1. The primitive data types in JavaScript are number, string, boolean, undefined, and null.

Number = numeric values like integers and decimals
String = text enclosed in quotes
Boolean = true or false
Undefined = a variable declared but not assigned a value
Null = intentional absence of a value

It honestly surprised me that JavaScript is so "okay" with mixing types. Python usually makes you be very explicit/specific, so JavaScript feels very loose with types.
